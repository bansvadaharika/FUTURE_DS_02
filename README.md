# Customer Retention & Churn Analysis Dashboard

## 📌 Project Overview
This project analyzes customer churn using the IBM Telco Customer Churn dataset. An interactive dashboard was created in Microsoft Excel using Pivot Tables, Pivot Charts, and KPI calculations to identify customer retention patterns and churn trends.

---

## 📂 Dataset
- IBM Telco Customer Churn Dataset
- Total Records: **7,043 Customers**

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Analysis
- Dashboard Design

---

## 📊 Key Performance Indicators (KPIs)
- Total Customers: **7043**
- Churned Customers: **1869**
- Active Customers: **5174**
- Churn Rate: **26.54%**
- Retention Rate: **73.46%**
- Average Monthly Charges: **64.76**
- Average Tenure: **32.37 Months**

---

## 📈 Dashboard Visualizations
- Customer Churn Distribution
- Churn by Contract Type
- Churn by Internet Service
- Churn by Payment Method
- Churn by Gender

---

## 💡 Key Insights
- Overall customer churn rate is **26.54%**.
- Customers with **Month-to-Month** contracts have the highest churn.
- Customers with **Two-Year** contracts have the lowest churn.
- Customers using **Electronic Check** show higher churn.
- **Fiber Optic** customers churn more than DSL customers.
- Customer churn is nearly equal across male and female customers.
